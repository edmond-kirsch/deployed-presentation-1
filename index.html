Cross-browser compatibility and testing
Why it happens?
  • Sometimes browsers contain bugs, or implement features differently.
  • Browsers have varying degrees of support for modern technologies.
    How to fix it?
  • Use semantic HTML.
  • Apply CSS wisely
  • Use polyfills.
  • Conduct thorough testing.
What exactly need to do?
Validate HTML and CSS with w3c validator and css validator
